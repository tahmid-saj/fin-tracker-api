# fin-tracker-api

API for planning expenses, banking, investments, savings, insurances and retrieving responses from a chatbot, getting historic market data, calculations from useful finance tools and more. Developed with Express, GraphQL, MongoDB, OpenAI API, Redis, PostgreSQL.
